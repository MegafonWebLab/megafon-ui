# TextField

## Properties

<Props of={TextField} />

### Classes property keys
field        | description
-------------|----------
*input*       |  Class that is passed to the input

## Basic usage

<Playground>
    <TextFieldWrapper>
        <TextField name="name" />
    </TextFieldWrapper>
</Playground>

## Required field

<Playground>
    <TextFieldWrapper>
        <TextField name="name" required label="Phone number" />
    </TextFieldWrapper>
</Playground>

## Mask usage

<Playground>
    <TextFieldWrapper>
        <TextField
            placeholder="+7 (999) 999-99-99"
            mask="+7 (999) 999-99-99"
            maskchar="_"
            label="Phone number"
            id="2"
            inputMode="numeric"
            classes={{input: 'parent__inputClass'}}
        />
    </TextFieldWrapper>
</Playground>

## Error usage
Ð¡licking on cross icon clears the field by default. This behavior won't work with defined custom icon or onCustomIconClick handler.

<Playground>
    <TextFieldWrapper>
        <TextField
            verification="error"
            noticeText="Name is required"
        />
    </TextFieldWrapper>
</Playground>

## Custom icon and handler
<Playground>
    <TextFieldWrapper width={550}>

            verification="error"

        />

            customIcon={ <CustomIcon style={{width: "32", height: "32", display: "block", userSelect: "none", fill: "#000000"}} /> }

        />
     </TextFieldWrapper>
</Playground>

# Textarea view
The **multiline** property set to **true** changes TextField to textarea-like view.

## Basic usage

<Playground>
    <TextFieldWrapper>
        <TextField name="name" multiline placeholder="Multiline"/>
    </TextFieldWrapper>
</Playground>
