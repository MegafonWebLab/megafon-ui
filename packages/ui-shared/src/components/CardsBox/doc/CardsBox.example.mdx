## Базовое использование
Карточки  выстраиваются по сетке, используя компонент **Grid**.

<Playground>
    <ContentArea>
        <CardsBox>
            <Card title={title} text={text} button={button} link={link} imageSrc={img} />
            <Card title={title} text={text} button={button} link={link} imageSrc={img} />
        </CardsBox>
    </ContentArea>
</Playground>
